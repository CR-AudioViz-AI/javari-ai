# Team Collective Intelligence Engine

## Purpose
The Team Collective Intelligence Engine is designed to compute consensus scores and collective decisions based on insights generated by various AI agents. This engine integrates event handling, agent management, and decision-making to facilitate effective team dynamics.

## Usage
To utilize the Team Collective Intelligence Engine, instantiate the `ConsensusCalculator` class with a configuration object, and use its methods to analyze insights from AI agents to derive collective decisions and consensus data.

### Example
```typescript
import { ConsensusCalculator } from './src/modules/team-intelligence/collective-engine';

const consensusConfig = {
  minimumParticipants: 3,
  agreementThreshold: 0.7,
  confidenceWeighting: true,
  timeDecayFactor: 0.5,
  convergenceTimeout: 30000
};

const calculator = new ConsensusCalculator(consensusConfig);

// Add methods to collect insights and calculate consensus
```

## Parameters/Props

### AgentInsight
- **agentId**: string - Unique identifier of the agent.
- **agentType**: string - Type of the agent (expert, specialist, etc.).
- **timestamp**: number - Time of insight generation (in milliseconds since epoch).
- **content**: string - The insight text.
- **confidence**: number - Confidence level of the insight (0 to 1).
- **metadata**: Record<string, any> - Additional data related to the insight.
- **tags**: string[] - Tags associated with the insight.
- **weight**: number - Weight assigned to the insight.

### ConsensusData
- **topic**: string - Subject of consensus.
- **agreementScore**: number - Final score representing the level of agreement (0 to 1).
- **participantCount**: number - Number of agents contributing insights.
- **majorityView**: string - The prevailing perspective among agents.
- **minorityViews**: string[] - Alternative opinions that did not reach consensus.
- **convergenceTime**: number - Time taken to reach consensus (in milliseconds).
- **conflictAreas**: string[] - Aspects where disagreements arose.

### CollectiveDecision
- **id**: string - Unique identifier of the decision.
- **topic**: string - Topic related to the decision.
- **decision**: string - The final decision made.
- **confidenceScore**: number - Confidence in the decision (0 to 1).
- **consensusScore**: number - Score representing the consensus strength.
- **participatingAgents**: string[] - IDs of agents contributing to the decision.
- **supportingInsights**: AgentInsight[] - Insights backing the decision.
- **dissendingViews**: AgentInsight[] - Insights against the decision.
- **timestamp**: number - Time when the decision was made.
- **validationScore**: number - Score assessing the decision's validity.
- **risks**: string[] - Identified risks associated with the decision.
- **recommendations**: string[] - Suggested actions stemming from the decision.

### ConsensusConfig
- **minimumParticipants**: number - Minimum number of agents needed to form a consensus.
- **agreementThreshold**: number - Minimum score for agreement to be considered valid.
- **confidenceWeighting**: boolean - Indicates whether to weight insights by their confidence.
- **timeDecayFactor**: number - Decay factor for time-related scoring.
- **convergenceTimeout**: number - Time limit for achieving consensus before timeout.

### WeightingConfig
- **expertWeight**: number - Weight for expert agents' insights.
- **specialistWeight**: number - Weight for specialist agents' insights.
- **generalistWeight**: number - Weight for generalist agents' insights.
- **experienceMultiplier**: number - Multiplier based on agent experience.
- **accuracyMultiplier**: number - Multiplier based on insight accuracy.
